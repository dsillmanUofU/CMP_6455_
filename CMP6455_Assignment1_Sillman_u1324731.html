SUBJECT: Workflow for Salt Lake County crash data by census tract



In order to better analyze the relationship between community characteristics and various types of crashed in Salt Lake County, it is necessary to begin with a better understanding of the geographical distribution of crashes. In order to demonstrate this geographical distribution, a choropleth map was created using Census Tracts and crash data. This memo will address the workflow by which junior analysts created said map, a draft of the map, and some preliminary observations that were derived from the finished product.


Workflow:


The flow chart above portrays the process for creating the choropleth map.


Data sources included Utah.gov Open Data Catalog for crash data, Utah Automated Geographic Reference Center for county boundaries (via services.1.arcgis.com) and Census Tract boundaries. All of these data sets were state-wide. The first step was to collect data from these sources. Data for crashes in Utah from 2015-2019 were available in CSV form and had to be projected using the geometric information in the data set. While projecting the data, it is necessary to choose the projection coordinate system to be used. All coordinate projections were changed to EPSG: 3566, including crash data.


After each layer was exported as a new layer with the correct coordinate system, data had to be filtered to Salt Lake County from the state data sets. In order to accomplish this, the Salt Lake County shapefile

was filtered via attribute. The selection was then exported and the state-wide data was removed from the map. Using this layer, it is now possible to filter crash data and census tracts by location.


Beginning with census tract data, a selection by location within the new Salt Lake City layer and manually adding the census tracts that are immediately within the county boundary is followed by exporting the selected components and removing the original layer. This will reduce the number of census tracts to those within Salt Lake County. After this layer has been filtered, the crash data is selected by location within and intersecting the Salt Lake County layer.


The two new layers are then joined using the special join summary tool. The operation for the join is “count”, the crash data is being joined to the census tract data. After the new layer is created from the join, it is possible to remove the census tract layer, it is also possible to remove the county layer. Using the properties menu for the new join layer, it is possible to create a categorized symbology. The Crash ID count was used as the value that was categorized. The mode used was Natural Breaks (Jenks).


Finally, it is important to have some context for the census tracts, and in particular it is important to have the roadways visible in order to better understand the patterns of crashes throughout Salt Lake County. Open Streets basemap was used at a low transparency to give this context.


Map Characteristics:



The choropleth map above shows the joined layer after it had been categorized. Because the crash data set was so large and it is difficult to differentiate between data, the new layer can give the viewer more

information about where the highest concentration of crashes occurred. The map demonstrates that there are more crashes along Interstate 15, near the Salt Lake City International Airport, in canyon roadways, and clustered throughout the county. Using the natural breaks classification method helps to reduce variance within clusters, meaning that the clusters presented by different colors have fewer outliers than if a different method of classification was used. The census tracts with the highest number of crashes are grouped with much larger intervals than the census tracts with the lowest number of crashes.


A potential limitation of this visualization is that there is no control for the amount of space that is included within each census tract. For example, the tract that include Little Cottonwood Canyon is far larger than several of the census tracts in the valley. Because the crash data is a count and not a rate, it may not be representative of the relative risk of driving in certain areas. However, in the case of Little Cotton Wood Canyon, there is a plausible causal relationship between the risk of the main roadway and the amount of traffic and the high number of crashes here.


Conclusion:


This map is an important component of the assessment requested by Salt Lake County. It is important to first demonstrate the Census Tracts that have a high number of crashes and the risks associated with this in order to inform the further question of equity.
